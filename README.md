# Face Recognition using PCA + ANN

## Project Title
Implementation of PCA with ANN algorithm for Face Recognition

## Description
A face recognition system built from scratch using:
- PCA (Eigenfaces) for feature extraction
- ANN (Neural Network) for classification

## Files
- main.py — Run this to execute everything
- pca.py — Manual PCA implementation
- ann_model.py — ANN built from scratch
- train.py — Training pipeline
- test.py — Testing + evaluation
- utils.py — Helper functions

## How to Run
pip install -r requirements.txt
python main.py
